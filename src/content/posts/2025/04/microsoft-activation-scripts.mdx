Microsoft Activation Scripts (MAS) 是一个开源工具，帮助用户激活 Windows 和 Office。它支持多种激活方式，包括永久和临时方案，适用于 Windows 7 到 11 以及 Office 2013 到 365。无论是个人还是企业用户，遇到激活问题时，MAS 都能提供简单可靠的解决方案。

## Microsoft Activation Scripts 使用教程

<Notice type="note">因 YouTube 的《社区准则》，无视频教程。</Notice>

### 下载

MAS 提供两种下载方式：

1. **PowerShell 方法（推荐）**  
    - 打开 PowerShell 或终端（右键点击开始菜单选择 PowerShell 或终端）。
    - 输入以下命令并按回车：
    ``` powershell
    irm https://get.activated.win | iex
    ```
    - 如果上述命令无法使用，可尝试备用命令：
    ``` powershell
    irm https://massgrave.dev/get | iex
    ```
    - 下载完成后，您将看到激活选项。

2. **传统方法**  
    - 从以下链接下载 ZIP 文件：
    ``` http
    https://github.com/massgravel/Microsoft-Activation-Scripts/archive/refs/heads/master.zip
    ```
    - 解压 ZIP 文件后，进入名为 `All-In-One-Version` 的文件夹。
    - 找到并运行 `MAS_AIO.cmd` 文件。

### 安装

MAS 不需要复杂安装，只需运行下载的脚本即可开始使用。对于 PowerShell 方法，直接执行上述命令；对于传统方法，运行解压后的脚本文件即可。

### 使用说明

1. **选择激活选项**  
    在 PowerShell 或传统方法中运行脚本后，根据提示选择激活方式：
    - 按 `1` 激活 Windows（HWID 方法）。
    - 按 `2` 激活 Office（Ohook 方法）。

2. **故障排除**  
    如果脚本未能正常运行，请检查以下几点：
    - 确保 URL 来源安全可靠。
    - 如果 PowerShell 方法失败，可尝试传统方法。
    - 查看脚本中的蓝色文字提示以获取故障排除步骤。
