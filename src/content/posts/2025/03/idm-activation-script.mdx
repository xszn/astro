IDM Activation Script 是一款开源激活脚本，可冻结试用期，实现永久免费使用。它支持 Windows 7 至 11 系统，通过重置试用期和锁定注册表实现激活，且更新后激活信息依然有效。用户可通过命令行或图形界面操作，简单易用，提升 IDM 使用体验。

## IDM Activation Script 使用教程

<Notice type="note">视频教程请移步 YouTube 观看《[IDM Activation Script 使用教程](https://youtu.be/aj9LYo5WzrQ)》。</Notice>

### 下载

``` powershell
iex(irm is.gd/idm_reset)
```

### 安装

<Tabs client:load>
    <Tab name="PowerShell">
        右键点击 `开始` 菜单，选择管理员权限的 PowerShell，复制粘贴命令后按回车运行脚本。
    </Tab>
    <Tab name="命令脚本">
        找到下载的 `IAS.cmd` 文件，右键选择 `以管理员身份运行` 来执行脚本。

</Tabs>

**推荐方法**

在“IDM Activation Script”窗口中按提示操作，建议按 `2` 选择“Freeze Trial”选项以锁定 30 天试用期，实现永久免费使用。

**备用方法**

从 GitHub 下载 IDM Activation Script，右键以管理员身份运行 `IAS.cmd` 文件，按 `2` 选择“Freeze Trial”冻结试用期。

IDM Activation Script 是一个实用的工具，可以帮助用户免费使用 IDM。按照上述步骤操作，即可轻松冻结 IDM 的试用期。建议使用 PowerShell 方法，并定期更新脚本以获得最佳效果。
